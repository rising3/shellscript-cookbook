## ターゲットのシェルについて

Linuxに標準でインストールされており、広く利用されているbashをターゲットにする。

**bashの特徴:**
- bashは、Bourne-again shellのこと
- bashは、Bourne Shellと互換性がある
- bashは、POSIXシェルの1つ(POSIXの仕様にほぼ準拠)
- ユーザーフレンドリーな操作性とスクリプトの柔軟性が高い

:::note[shの過去と現在]
shはBourne Shellと(間違って)解釈されることが多いが、Bourne Shellは昔のUNIXで利用されていた古いシェルの名前である。
Bourne Shellは終焉を迎えようとしており、現在はBourne Shellはほとんど利用されておらず、後継のPOSIXシェルが広く利用されている。
Linuxでは、/bin/shはashやdashなどのシェルへのシンボリックリンクになっており、Linuxのディストリビューションによって異なる。
:::

## 動作確認の環境について

筆者の都合上、次の環境で動作確認を行っている。

- Fedora Linux 40
- Manjaro Linux
- macOS Sequoia 15.0.1 + Homebrew
- bash 5系

なおサンプルコードについては、bash 4系以上で動作することを心掛けるが、動作しなかった場合は修正したソースコードをPull Requestしてほしい。

:::note[macOSのbashのバージョン]
macOSのbashのバージョンは、ライセンスとの兼ね合いで最新OSでも3系がインストールされている。
3系は2007年にリリースされたものでかなり古いバージョンとなる。
bash自体は何年も進化し続けて、2019年初めにはv5.0になっている。
[Homebrew](https://brew.sh/ja/)をインストールすれば、macOSでも最新のbashをインストールして利用することができる。
:::
