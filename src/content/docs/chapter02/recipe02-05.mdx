&& や || は、複数のコマンドを連結して実行するための記号である。
&& はAND演算で || はOR演算である。

&& (AND演算)の場合は、`コマンド1 && コマンド2` と書くことができ、コマンド1を実行して完了ステータスが 0 である場合、コマンド2を実行する。

**cd コマンドが成功したら、ls コマンドを実行する:**
```bash
» cd ~ && ls 
```

|| (OR演算)の場合は、`コマンド1 || コマンド2` と書くことができ、コマンド1を実行して完了ステータスが 0 以外である場合、コマンド2を実行する。

**cd コマンドが失敗したら、mkdir コマンドを実行する:**
```bash
» file=~/work
» cd "${file}" || mkdir "${file}"
```

`[` コマンドでも && (AND演算)または || (OR演算)を使用できる。

```bash
if [ 条件式1 ] && [ 条件式2 ]; then
  条件式1 と 条件式2 のどちらも真の場合の処理
if

if [ 条件式1 ] || [ 条件式2 ]; then
  条件式1 または 条件式2 のどちらかが真の場合の処理
if
```
