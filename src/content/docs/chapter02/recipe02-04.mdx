`[` コマンドや `test` コマンドで使用する演算子を紹介する。

演算子の紹介に入る前に`[` コマンドや `test` コマンドの書き方について改めて説明する。

## `[` コマンド

`[` コマンドは、`[` で開始し、 `]` で終了する。
また`[]`の中の値は、すべてスペースで区切る必要がある。

```bash
[ "${1}" = 'yes' ] # [ ] の中の値はスペースで区切ること
echo "$?"
```

## `test` コマンド 

`test` コマンドに渡す値は、すべてスペースで区切る必要がある。

```bash
test "${1}" = 'yes'
echo "$?"
```

## 評価演算子

### 文字列の評価演算子 

演算子 | 使用例 | 説明
:---: | :---: | ---
 = | s1 = s2 | s1 と s2 は等しい
 == | s1 == s2 | = と同じ
 != | s1 != s2 | s1 と s2 は等しくない
 \< | s1 < s2 | s1 は s2 よりも、辞書順に前にある
 \> | s1 > s2 | s1 は s2 よりも、辞書順に後にある
 -n | -n s1 | s1 は空文字列ではない
 -z | -z s1 | s1 は空文字列である

### 整数の評価演算子 

演算子 | 使用例 | 説明
:---: | :---: | ---
 -eq | i1 -eq i2 | i1 と i2 は等しい
 -ne | i1 -ne i2 | i1 と i2 等しくない
 -lt | i1 -lt i2 | i1 は i2 より小さい (\<)
 -le | i1 -le i2 | i1 は i2 以下 (=\<)
 -gt | i1 -gt i2 | i1 は i2 より大きい (\>)
 -ge | i1 -ge i2 | i1 は i2 以上 (=\>)

### ファイル属性の評価演算子

演算子 | 使用例 | 説明
:---: | :---: | ---
 -a | -a ファイル名 | ファイルが存在する
 -b | -b ファイル名 | ファイルが存在し、特殊ファイル（ブロック）である
 -c | -c ファイル名 | ファイルが存在し、特殊ファイル（キャラクター）である
 -d | -d ディレクトリ名 | ディレクトリが存在する
 -e | -e ファイル名 | -a と同じ
 -f | -f ファイル名 | ファイルが存在し、普通のファイルである

### 結合演算子

演算子 | 使用例 | 説明
:---: | :---: | ---
 -a | 条件式1 -a 条件式2 | 条件式1 と 条件式2 が真の場合に 真 (AND)
 -o | 条件式1 -o 条件式2 | 条件式1 もしくは 条件式2 が真の場合に 真 (OR)
 !  | ! 条件式 | 条件式の真偽を逆にする
 () | ( 条件式1 -a 条件式2 ) -o 条件式3 | 条件をグルーピングする
